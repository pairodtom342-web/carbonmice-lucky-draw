# 🎁 carbonMICE Lucky Draw

เว็บจับรางวัลผู้โชคดีสำหรับงาน Event — ออกแบบให้แชร์ขึ้น **จอ LED ขนาดใหญ่** ในงาน

## 🔗 ใช้งาน
https://pairodtom342-web.github.io/carbonmice-lucky-draw/

## วิธีใช้ (MC / เจ้าหน้าที่หน้างาน)
1. กด **⚙️ ตั้งค่า / รายชื่อ** → ใส่ชื่องาน, โลโก้งาน (ไม่บังคับ), ชื่อรางวัลที่กำลังจับ
2. วางรายชื่อผู้เข้าร่วม (1 คน/บรรทัด — วางจาก Excel/Google Sheet ได้เลย, ใส่หน่วยงานคั่นด้วย `,` หรือ Tab) หรืออัปโหลดไฟล์ CSV
3. กด **บันทึก & เริ่มจับรางวัล** → กด **⛶ เต็มจอ** สำหรับขึ้นจอ LED
4. กด **🎲 สุ่มรายชื่อ** (หรือกดปุ่ม Space) → ชื่อจะไล่สุ่มแล้วหยุดที่ผู้โชคดี
5. เลือกผล:
   - **✅ อยู่ในงาน — รับรางวัล** → บันทึกเป็นผู้ได้รางวัล (มี effect ฉลอง 🎉) และตัดออกจากกล่อง
   - **❌ ไม่อยู่ — ตัดชื่อออก** → ตัดออกจากกล่อง (ไม่ได้รางวัล)
6. เปลี่ยน "ชื่อรางวัล" ที่หน้าตั้งค่าได้ตลอด เมื่อขึ้นรางวัลชิ้นใหม่

## จุดเด่น
- ตัวนับสด: 🎯 ยังไม่ได้จับ · 🏆 ได้รางวัลแล้ว · 🚫 ไม่อยู่/ตัดออก · 👥 ทั้งหมด
- รายชื่อผู้ได้รางวัล (พร้อมชื่อรางวัล) เรียงล่าสุดบนสุด
- คนที่จับไปแล้ว (ได้รางวัล/ไม่อยู่) จะไม่ถูกสุ่มซ้ำ
- **↩️ ย้อนกลับ** แก้ผลรายการล่าสุดได้ (กันกดพลาด)
- **จำสถานะอัตโนมัติ** — รีเฟรช/ปิดจอแล้วเปิดใหม่ ข้อมูลไม่หาย (เก็บในเครื่องนี้)
- 🔄 เริ่มจับรอบใหม่ (คืนทุกคนเข้ากล่อง) · 🗑️ ล้างข้อมูลทั้งหมด
- ดีไซน์จอมืด คอนทราสต์สูง ตัวอักษรใหญ่ อ่านง่ายบนจอ LED · คงแบรนด์ carbonMICE

## หมายเหตุ
- ทำงานทั้งหมดในเบราว์เซอร์ (ไม่ต้องต่อเซิร์ฟเวอร์ / เล่นออฟไลน์ได้) ข้อมูลรายชื่อเก็บเฉพาะในเครื่องที่เปิด
- แนะนำเปิดบนคอมที่ต่อจอ LED โดยตรง แล้วกดเต็มจอ
- แนวนอน (16:9) เหมาะกับจอ LED · เปิดบนจอแนวตั้งก็ปรับ layout ให้อัตโนมัติ
